# 667 ЖҰМЫС ІСТЕ! Chrome Extension

Бұл кеңейтім уақытты босқа өткізетін сайттарға кіргенде, мотивациялық хабарлама көрсетеді.

## Мақсаты

Сіз таңдаған сайттарға кірген кезде, олар бұғатталып, орнына мотивациялық хабарлама көрсетіледі:  
**"667 ЖҰМЫС ІСТЕ. УАҚЫТТЫ БОСҚА ӨТКІЗБЕ."**

## Орнату

1. Репозиторийді жүктеңіз немесе көшіріңіз.
2. Chrome браузерінде `chrome://extensions` бетіне кіріңіз.
3. "Developer mode"-ды қосыңыз.
4. "Load unpacked" батырмасын басып, осы папканы таңдаңыз.

## Пайдалану

- Кеңейтімнің параметрлеріне кіріп (`Options`), бұғаттағыңыз келетін сайттарды енгізіңіз (мысалы: `youtube.com`).
- Бұғатталған сайтқа кіргенде, мотивациялық хабарлама көрсетіледі.

## Файлдар

- [`background.js`](background.js) — Бұғаттау ережелерін басқарады.
- [`options.html`](options.html), [`options.js`](options.js), [`options.css`](options.css) — Бұғатталған сайттарды басқару интерфейсі.
- [`motivational.html`](motivational.html), [`style.css`](style.css) — Мотивациялық хабарлама беті.
- [`manifest.json`](manifest.json) — Кеңейтімнің сипаттамасы.
